# Challenge

You have developed a very simple contacts list web application that allows users to view, add, update, and delete contact information. But it is not very fault-tolerant or user-friendly and it doesn't look very good. In this challenge, you will enhance the application by adding validation and error handling to improve the user experience. You will also improve the visual appearance of the application by adding CSS styles to make it more visually appealing.

## Task 1: Implement Validation

1. Add validation to the `insert.php` and `update.php` files to ensure that the user input is valid before inserting the data into the database.
2. Validate the following fields:
   - First Name (required, text input)
   - Last Name (required, text input)
   - Email (required, email input)
   - Phone (required, phone input)
3. Display appropriate error messages if the user input is invalid.

## Task 2: Implement Error Handling

1. Implement error handling in the application to display user-friendly error messages when an error occurs.
2. Handle missing GET parameters in the `update.php` and `delete.php` files.

## Task 3: Add CSS Styles

1. Create a new CSS file named `styles.css` and link it to all PHP files in the application.
2. Add CSS styles to improve the visual appearance of the application. You can style the following elements:
   - Page title
   - Heading
   - Navigation block
   - Form elements
   - Table elements

## Task 4: Create `index.php` File

1. Create a new PHP file named `index.php`.
2. Add a simple welcome message and a brief description of the application.
3. Include links to the `select.php`, `insert.php`, `update.php`, and `delete.php` files.
