# PHP Database Integration

Lab exercises on using PHP with a database.

Details on the configuration and usage of using Codespaces for this lab are in the [`codespaces.md` document](codespaces.md).

## Exercises

In this lab, you will work with PHP and MySQL to create a simple contacts list web application that interacts with a database. The lab exercises are as follows:

1. [Exercise 1 - Database](exercise1.md): Create a database and a table to store data.
2. [Exercise 2 - Connect to Database](exercise2.md): Connect to the database using PHP and retrieve data from the `contacts` table.
3. [Exercise 3 - INSERT Data](exercise3.md): Insert data into the `contacts` table using PHP, using a form to collect user input.
